# Rock Paper Scissors Game

This is a simple Rock, Paper, Scissors game built with HTML, CSS, and JavaScript.

## How to Play

1.  Open the `index.html` file in your web browser.
2.  Click on one of the three choices: rock, paper, or scissors.
3.  The computer will make its choice, and the winner of the round will be displayed.
4.  The first player to reach 3 points wins the game.
5.  A modal will appear to announce the winner, and you can click the "Play Again" button to start a new game.

## Features

*   Simple and intuitive interface
*   Keeps track of the score
*   Announces the winner of the game
*   Confetti effect when the user wins
*   "Play Again" functionality

## Files

*   `index.html`: The main HTML file that structures the game.
*   `style.css`: The CSS file that styles the game.
*   `script.js`: The JavaScript file that contains the game logic.
